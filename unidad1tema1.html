TEMA 1- Partes de la computadora

Hardware y Software

Hardware: Es la parte física de la computadora, es decir todo lo que la compone que hace que el software funcione

Algunas de las partes de este son:

◉ Teclado

◉ Unidad central d Proceso CPU

◉ Monitor

◉ Ratón

◉ Disquetera

◉ CD-ROM

◉ Placa Madre

◉ Memoria RAM

◉ Complemento para enfrentamiento

◉ Fuente eléctrica

◉ Puertos de Comunicación

◉ Disco duro

Software: Programas, instrucciones y lenguajes que permiten a la computadora realzar determinadas tareas. Es el conjunto de programas y reglas informáticas que hacen posible el funcionamiento del equipo. Aporta las operaciones.
